# Admin Dashboard Desktop Application 



### Technology Uses
- Electron
- ReactJS
- Antd
- react-router-dom
- tailwindcss
- react-pro-sidebar
- redux
- and much more...



### Run 
```shell
# To run development environment
npm run start [for React] 
npm run electron [for open electron window]

Note: 
```

### Build
```shell

# To build react application
npm run build

# To build Electron for windows
npm run make:win
```
